# HaiBar

Native macOS menu bar app for searching books on [raamatuhai.ee](https://raamatuhai.ee).

Search for books, browse latest listings, and see popular picks — all from your menu bar without opening a browser. Click the menu bar icon or press **Cmd+Shift+B**.

## Download

**[HaiBar-1.1.0.dmg](https://github.com/alvarlaigna/haibar-releases/raw/main/releases/HaiBar-1.1.0.dmg)** (770 KB)

Requires macOS 13+ (Ventura).

### Installing

1. Open the downloaded `.dmg` file
2. Drag **HaiBar** to Applications
3. Since the app is not notarized, right-click HaiBar.app and select **Open**, then click **Open** in the dialog
   - Alternatively: System Settings → Privacy & Security → **Open Anyway**

## Features

- Search books by title or ISBN
- Browse latest listings and popular picks
- See prices with one click
- Global hotkey: **Cmd+Shift+B**
- Lightweight menu bar app — no Dock icon
